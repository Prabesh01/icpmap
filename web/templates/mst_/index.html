Bulk Download MST Contents

One-Time Setup:

1. Go to the directory where you want to download contents and run,


git init
git remote add origin -f https://oauth2:{{ token }}@gitlab.com/Prabesh01/mst.git
git config core.sparseCheckout true

2. Now, edit the .git/info/sparse-checkout file and add the folders you want to download. Skip this if you want to download all files.


y2/
OR, 

y2/CS5002 Software Engineering
y2/CS5053 Cloud Computing and the Internet of Things

3. Finally run,


git pull origin main

That's all!

Start Syncing:

Run this command everytime you want to fetch new uploaded contents:


git pull
